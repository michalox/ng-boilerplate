Install Node.js and then:
```sh
$ git clone git://github.com/joshdmiller/ng-boilerplate
$ cd ng-boilerplate
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt watch
```
Finally, open `file:///path/to/ng-boilerplate/build/index.html` in your browser.
